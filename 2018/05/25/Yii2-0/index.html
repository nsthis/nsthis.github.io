文件目录

yii框架有2个版本，到时候具体看你选择

基础版：

basic 目录下的各个文件夹：

commands 控制台

config 配置文件

console.php 控制台配置

db.php 数据库连接配置

params.php 项目中的变量配置

web.php web应用配置

controllers 控制器层

models 模型层

runtime 运行时的临时文件目录（缓存文件和日志文件），需要写权限

vendor 存放 Yii 的源码和通过 composer 安装的项目依赖库

views 视图层

web 域名指向的目录，包含入口文件，可存放css、js、img、font等静态资源

高级版：

advanced 目录下的各个文件夹：

backend 后台应用

common 公共目录

console 控制台应用

environments 环境目录（开发环境和生产环境）

frontend 前台应用

vendor 存放 Yii 的源码和通过 composer 安装的项目依赖库
